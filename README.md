## Data

Wallpapers are saved in:

`<app_folder>/userdata/library/wallpapers`

<br>
<br>

## Building

1.  Install the **[.NET 5 SDK]**

2. Navigate to project folder

3. Build the project with:

    ```shell
    dotnet build
    ```
